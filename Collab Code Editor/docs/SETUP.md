# CollabCode - Setup Guide

This guide will help you set up and run the CollabCode collaborative code editor on your local machine.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Web Browser**: Chrome 60+, Firefox 55+, Safari 12+, or Edge 79+
- **Node.js** (optional): For using the development server
- **Git**: For version control (optional)
